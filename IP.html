Examples

Formatted Python Notes

Maintain professional coding notes with comments, algorithms, and clear output formatting. 

# Problem: Find maximum element from a list

# Algorithm:
# 1. Take list input
# 2. Assume first element is maximum
# 3. Compare remaining elements
# 4. Print final maximum value

numbers = [23, 45, 67, 12, 89]
maximum = numbers[0]

for num in numbers:
    if num > maximum:
        maximum = num

print("Maximum Number:", maximum)

# Output:
# Maximum Number: 89
